# SAP UI5 Walkthrough

SAP UI5 Walktrough course found [here](https://sapui5.hana.ondemand.com/#/topic/3da5f4be63264db99f2e5b04c5e853db).

## Run the application

1. Clone the repository

2. Install dependencies

```
npm install
```

3. Run the local server

```
ui5 serve
```
